# My GitHub Pages Site

This is a minimal static site (HTML/CSS/JS) ready to publish with GitHub Pages.

How to use:
1. Clone the repository.
2. Edit the files (index.html, style.css, script.js).
3. Push to GitHub and enable Pages (see below).

GitHub Pages:
- For a project site: In repo Settings â†’ Pages, choose branch `main` (or `gh-pages`) and folder `/ (root)` or `/docs`.
- For a user site (username.github.io): create a repo named `username.github.io` and push to `main` root.

Local preview:
- Use the Live Server VS Code extension or run `python -m http.server` (Python 3) in the folder and open http://localhost:8000.
